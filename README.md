# lavadora-logs
Aquest Portal IT és una eina corporativa "Zero-Footprint" (100% RAM) que permet a l'Administració anonimitzar dades sensibles amb IA local, gestionar PDFs (unir/extreure/rotar), comprimir imatges per al registre i aplicar segells digitals. Garantim privacitat total: cap dada surt mai del servidor municipal.
